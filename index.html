Modul 2. HTML

CSS Methods

1. Inline CSS

Metode Inline CSS diterapkan langsung dalam element HTML menggunakan atribut style. Metode ini cocok untuk styling secara cepat, namun tidak efektif untuk hal yang lebih besar.

<p style="color: blue; font-size: 16px;">This is an inline-styled paragraph.</p>

This is an inline-styled paragraph.

2. Internal CSS

Metode Internal CSS ditulis di tag <head>pada dalam tag <style> di dokumen HTML <head> section of an HTML document. Efektif untuk apply style pada suatu page tanpa mempengaruhi page yang lain.

<!DOCTYPE html>
  <html>
  <head>
  <style>
    p {
        color: green;
        font-size: 18px;
    }
  </style>
  </head>
  <body>
  <p>This is an internally-styled paragraph.</p>
  </body>
  </html>

Kalimat di-style menggunakan internal CSS.

3. External CSS

Metode External CSS ditulis di file berbeda dengan ekstensi file .css dan dihubungkan ke dokumen HTML menggunakan tag <link>. Metode ini yang sering digunakan ketika styling lebih kompleks.

<!DOCTYPE html>
  <html>
  <head>
  <link rel="stylesheet" href="styles.css">
  </head>
  <body>
  <p>This is an externally-styled paragraph.</p>
  </body>
  </html>

And in styles.css:

p {
  color: red;
  font-size: 20px;
  }

Kalimat ini menggunakan external css

HTML Layout with Semantics

1. <header>

Element <header> header biasanya berisi konten seperti navigasi, ataupun logo.

<header>
  <h1>Welcome to Our Website</h1>
  <nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
  </nav>
  </header>

2. <nav>

Element <nav> biasanya berisi link-link navigasi.

<nav>
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#services">Services</a></li>
    <li><a href="#contact">Contact</a></li>
  </ul>
  </nav>
